
## 프롬프트 요청 시 주의사항 숙지
```
- 비밀번호, API 키 등 민감한 개인정보가 포함된 컨텐츠에 대해 경고 요청
```



### 구조화된 프롬프트 예시
```
[상황/맥락]: 웹 개발 프로젝트에서 사용자 인증 시스템을 구현하고 있습니다.
[구체적인 질문]: React와 Node.js를 사용하여 JWT 기반 인증을 어떻게 구현할 수 있나요?
[원하는 결과물 형태]: 코드 예시와 단계별 설명
[제약사항이나 특별한 요구사항]: TypeScript 사용, 보안 모범 사례 포함
```
